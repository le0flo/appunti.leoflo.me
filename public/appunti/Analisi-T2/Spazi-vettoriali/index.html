    Prodotto cartesiano tra due insiemi #

    $$ \mathbb{A}_1 \times \mathbb{A}_2 = \{ (a_1, a_2) : a_1 \in \mathbb{A}_1, a_2 \in \mathbb{A}_2 \} $$

    Coincidenza di due coppie cartesiane #

    $$ (a_1, a_2) = (a_3, a_4) \implies a_1 = a_3, a_2 = a_4 $$

    Insiemi coincidenti #

    $$ \mathbb{A}_1 \times \dots \times \mathbb{A}_n = \mathbb{A}^n $$$$ \mathbb{R}^n = \{ (x_1, \dots, x_n) : x_1 \in \mathbb{R}, \dots, x_n \in \mathbb{R} \} $$

    Somma in $\mathbb{R}^n$ #

    $$ (\mathbb{R}^n \times \mathbb{R}^n \to \mathbb{R}^n) = (x \times y \to x + y \in \mathbb{R}^n) $$

    e.g.

    • $n=3, x=(1, \pi, -1), y=(0,-\pi, 2), x+y=(1,0,1)$
    • $x=(x_1, \dots, x_n), y=(y_1, \dots, y_n), x+y=(x_1+y_1, \dots, x_n+y_n)$

    Proprietà della somma #

    Dati: $x,y,z \in \mathbb{R}^n, \lambda, \micro \in \mathbb{R}$

    1. $y + x = x + y$
    2. $(x+y)+z = x+(y+z)$
    3. $0 = (0, \dots, 0), \forall x \in \mathbb{R}^n, x+0 = 0+x = x$
    4. $\forall x \in \mathbb{R}^n, \exists -x : x+(-x) = (-x)+x = 0$
    5. $(\lambda + \micro)x = (\lambda x) + (\micro x)$
    6. $\lambda(x+y) = \lambda x + \lambda y$
    7. $\lambda(\micro x) = (\lambda \micro)x$
    8. $\forall x \in \mathbb{R}^n, 1x = x$

    Prodotto scalare in $\mathbb{R}^n$ #

    $$ x * y = = x_1y_1 + \dots + x_ny_n = \sum^{n}_{j=1}x_jy_j $$

    Operazione di tipo $\mathbb{R}^n \times \mathbb{R}^n\to \mathbb{R}$.

    • $x*y = 0 \implies$ $x$ e $y$ sono ortogonali
    • $x*y = ($lunghezza di $x)*($lunghezza di $y) * cos(0)$

    Proprietà del prodotto scalare #

    Dati: $x,y,z \in \mathbb{R}^n, \lambda \in \mathbb{R}$

    1. $(x+y)z = x*z + y*z$
    2. $y*x = x*y$
    3. $(\lambda x)y + \lambda(x*y) = x(\lambda y)$
    4. $x*x \geq 0 \implies x=(x_1, \dots, x_n), x*x=(x_1^2 + \dots x_n^2)$
    5. $x*x = 0 \implies x=(0, \dots, 0)$

    Disuguaglianza di Cauchy-Schwarz #

    $$ |x*y| \leq \sqrt{(x*x)} * \sqrt{(y*y)} $$

    $\sqrt{(x*x)} = (x_1^2, \dots, x_n^2)^{\frac{1}{2}} =$ lunghezza del vettore $x$ o norma. La disuguaglianza ci afferma che il valore assoluto del prodotto tra due vettori $x$ e $y$ è minore o uguale rispetto al prodotto tra le loro lunghezze.

    Proprietà delle norme #

    Dati: $x,y \in \mathbb{R}^n, \lambda \in \mathbb{R}$

    1. $||x|| > 0 \implies x \neq 0$
    2. $||x|| = 0 \implies x = 0$
    3. $||\lambda x|| = |\lambda|*||x||$
    4. $||x+y|| \leq ||x|| + ||y||$

    Distanza in $\mathbb{R}^n$ #

    $$ d(x,y) = ||x*y|| $$

    Proprietà della distanza #

    Dati: $x,y,z \in \mathbb{R}^n$

    1. $d(x,y) \geq 0$
    2. $d(y,x) = d(x,y)$
    3. Disuguaglianza triangolare: $d(x,z) \leq d(x,y) + d(y,z)$

    “Palla” #

    $$ B(x,r) = \{ y \in \mathbb{R}^n : d(x,y) < r \} $$
