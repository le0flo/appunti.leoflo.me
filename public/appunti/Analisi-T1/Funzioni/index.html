    Approssimazione lineare #

    $$ o(1) \implies \text{è una funzione che tende a 0 per } h \to 0 $$$$ g(h) = o(h^n) \text{ per } h \to 0 \implies \frac{g(h)}{h^n} \to 0 \text{ per } h \to 0 \implies \frac{g(h)}{h^n} = o(1) \text{ per } h \to 0 $$

    Punti critici #

    • Data una funzione derivabile $f(x)$, $x_0$ è un punto critico di essa se $Df(x_0) = 0$.
    • Data una funzione derivabile $f(x)$, i punti di massimo e minimo locale sono punti critici.

    Teorema di Rolle #

    $$ \exists \space c \in \space ]a,b[ \space : Df(c) = 0 $$

    Teorema di Weierstrass #

    $$ \exists \text{ un massimo ed un minimo di }f $$

    Teorema di Lagrange #

    $$ \exists \space c \in \space ]a,b[ \space : Df(c) = \frac{f(b)- f(a)}{b - a} $$

    Teorema De l’Hopital #

    $$ \lim_{x \to x_0} \frac{f(x)}{g(x)} = \lim_{x \to x_0} \frac{Df(x)}{Dg(x)} = L \in [-\infty,+\infty] $$

    Teorema di Cauchy #

    $$ \exists \space c \in \space ]a,b[ \space : (f(b) - f(a))Dg(c) = (g(b) - g(a))Df(c) \implies \frac{Df(c)}{Dg(c)} = \frac{f(b) - f(a)}{g(b) - g(a)} $$
